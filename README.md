# Bash Script Custom

## Description
This script is more for a personal use and preference. It has in it, my customized aliases, functions, starship conf for my .bashrc configuration

## What's in it
Aliases:
```bash
# system
alias c
alias cls
alias q
alias quit
alias pls
alias install
alias remove
alias update
alias src

# npm
alias nrs
alias nrd

# git
alias gst
alias gpl
alias gpu
alias gco
alias gcm
alias gaa
```

Functions:
```bash
# Install/Update Discord and add Vencord Client
vencord-discord()

# git custom function to fetch, check diff, add, commit and push. All in one function
git-push()
```

Configuration
```bash
# Starship.toml conf
starship init bash
```
